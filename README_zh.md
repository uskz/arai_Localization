# arai_Localization

"浣熊公寓探索游戏 (Arai-san Mansion Exploration Game: Incarnation)" 翻译数据仓库

## 📝 概要

本仓库用于进行“浣熊公寓探索游戏 〜Incarnation〜”多语言支持的翻译工作。由游戏制作者（Owner）负责仓库的管理。

---

## 🔧 参与方式（适用于熟悉 GitHub 的成员）

1. 请**克隆（Clone）`main` 分支**。
2. 请以任意名称创建您的**个人分支**用于翻译（例如：`translation-zh`, `fix-typo-en` 等）。
3. 更新翻译后的 CSV 或相关文件，并**提交（Commit）**。
4. 翻译完成后，请向 `main` 分支**创建 Pull Request（PR）**。

> [!NOTE]
> 由于翻译文件极易发生冲突（Conflict），PR 经审核通过后的合并（Merge）操作将由 Owner 统一进行。

## ⚠️ 翻译注意事项

* **严禁使用 Excel 编辑 CSV 文件**。因为 Excel 自动保存时会导致编码或格式发生意料之外的改变。
* **严禁在翻译文件中自行添加或删除行**。
* 请务必注意：翻译文本中若含有 `{...}` 或 `[...]` 等被括号包裹的部分，**请勿修改或删除**。否则会导致游戏内必要的信息无法正常显示。
* 原则上，Owner 会给予大家充分的信任，直接合并（Merge）翻译内容。
* 即使只修改或翻译一小部分内容也完全没有问题，非常欢迎。
* 可以使用机器翻译，但请在最后务必**进行人工校对与润色**。
