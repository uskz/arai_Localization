# arai_Localization

"아라이상 맨션 탐색 게임(Arai-san Mansion Exploration Game: Incarnation)" 번역 데이터 저장소입니다.

## 📝 개요

이 저장소는 '아라이상 맨션 탐색 게임 ~Incarnation~'의 다국어 지원을 위한 번역 작업을 진행하는 공간입니다. 게임 제작자(Owner)가 저장소를 관리합니다.

---

## 🔧 작업 방법 (GitHub 사용이 가능하신 분)

1. `main` 브랜치를 클론(Clone)해 주세요.
2. 작업용으로 원하는 이름의 **개인 브랜치**(예: `translation-ko`, `fix-typo-en` 등)를 생성해 주세요.
3. 번역한 CSV 및 관련 파일을 업데이트한 후 커밋(Commit)해 주세요.
4. 작업이 완료되면 `main` 브랜치를 향해 **Pull Request(PR)를 생성**해 주세요.

> [!NOTE]
> 번역 파일은 충돌(Conflict)이 발생하기 쉬우므로, PR 승인 후의 머지(Merge)는 Owner가 진행합니다.

## 작업 시 주의사항

* **Excel을 통한 CSV 편집은 금지**합니다. 의도치 않은 서식 변경이나 왜곡이 발생할 수 있기 때문입니다.
* 번역 파일의 **행을 추가하거나 삭제하는 것은 금지**합니다.
* 번역문 중 `{...}`나 `[...]` 등 괄호로 둘러싸인 부분은 변경하거나 삭제하면 필요한 정보가 표시되지 않으므로 주의해 주세요.
* 기본적으로 참여자분들의 번역을 신뢰하여 머지(Merge)합니다.
* 일부분만 편집하여 제출해 주셔도 괜찮습니다.
* 기계 번역을 사용하셔도 무방하나, 마지막에는 반드시 사람의 눈으로 검토를 부탁드립니다.
