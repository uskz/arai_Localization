# arai_Localization

"アライさんマンション探索ゲーム(Arai-san Mansion Exploration Game: Incarnation)" translation data repository

## 📝 概要

このリポジトリは、「アライさんマンション探索ゲーム 〜Incarnation〜」の多言語対応のための翻訳作業を行うための場所です。ゲーム制作者（Owner）がリポジトリ管理を行います。

---

## 🔧 作業方法（GitHub が使える方向け）

1. **`main` ブランチをクローン**してください。
2. 作業用に、任意の名前で**個人ブランチ**（例：`translation-ja`, `fix-typo-en` など）を作成してください。
3. 翻訳した CSV や関連ファイルを更新し、**コミット**してください。
4. 作業が完了したら、`main` ブランチに向けて **Pull Request（PR）を作成**してください。

> [!NOTE]
> 翻訳ファイルはコンフリクトが発生しやすいため、PR の承認後のマージは Owner が行います。

---

## 🧩 作業方法（GitHub が使えない方向け）

GitHub の操作が難しい場合は、以下の手順で作業をお願いします。

1. 翻訳用ファイルを下記リンクからダウンロードして編集してください。（右クリックから「リンク先を名前を付けて保存」を選択）
   * 📄 [翻訳ファイル](https://raw.githubusercontent.com/<ユーザー名>/<リポジトリ名>/main/<パス>/<ファイル名>)
2. 更新が終わったら、下記の翻訳ファイル投稿フォームからファイルをアップロードしてください。
   * 📬 [翻訳ファイル投稿フォーム](https://docs.google.com/forms/d/e/1FAIpQLSe4aPeQpaxzxw94yu07YhepkyXko5SqtVkr_kT7ryfplmN6HQ/viewform?usp=publish-editor)

---

## 作業上の注意点

- 翻訳ファイルへの行の追加、削除は禁止です。
- 翻訳文中の{hp}などは変更したり消してしまうと必要な情報が表示されなくなりますので気をつけてください。
- 基本的に翻訳を信用してマージします。
- 機械翻訳でも構いませんが、最後に人間の目でチェックをお願いします。
