# arai_Localization
"アライさんマンション探索ゲーム(Arai-san Mansion Exploration Game: Incarnation)" translation data repository

📝 概要
このリポジトリは、
「アライさんマンション探索ゲーム 〜Incarnation〜」の多言語対応のための翻訳作業を行うための場所です。ゲーム制作者（Owner）がリポジトリ管理を行います。

🔧 作業方法（GitHub が使える方向け）
①main ブランチをクローンしてください。
②作業用に、任意の名前で個人ブランチ（例：translation-ja, fix-typo-en など）を作成してください。
③翻訳した CSV や関連ファイルを更新し、コミットしてください。
④作業が完了したら、main ブランチに向けて Pull Request（PR）を作成してください。
※翻訳ファイルはコンフリクトが発生しやすいため、PR の承認後のマージは Owner が行います。

🧩 作業方法（GitHub が使えない方向け）
GitHub の操作が難しい場合は、
翻訳用ファイルをこちらからダウンロードして編集してください。(右クリックから「リンク先を名前を付けて保存」してください)
https://raw.githubusercontent.com/<ユーザー名>/<リポジトリ名>/main/<パス>/<ファイル名>

更新が終わったら、下記の翻訳ファイル投稿フォームから翻訳ファイルをアップロードしてください。
https://docs.google.com/forms/d/e/1FAIpQLSe4aPeQpaxzxw94yu07YhepkyXko5SqtVkr_kT7ryfplmN6HQ/viewform?usp=publish-editor
