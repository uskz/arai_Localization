# arai_Localization

"Arai-san Mansion Exploration Game: Incarnation" translation data repository

## 📝 Overview

This repository is dedicated to the multilingual localization of "Arai-san Mansion Exploration Game ~Incarnation~". The repository is managed and maintained by the game creator (Owner).

---

## 🔧 Workflow (For GitHub Users)

1. **Clone** the `main` branch.
2. Create a **personal branch** with any descriptive name for your work (e.g., `translation-ja`, `fix-typo-en`).
3. Update the translated CSV or related files, then **commit** your changes.
4. Once your work is complete, **create a Pull Request (PR)** targeting the `main` branch.

> [!NOTE]
> Since translation files are highly prone to merge conflicts, the Owner will handle the final merging after approving the PR.

---

If you are not familiar with GitHub operations, please follow these steps:

## Important Guidelines & Notes
- **Do not use Microsoft Excel to edit CSV files.** This is strictly prohibited as Excel can corrupt or alter data formatting unintentionally. Please use a text editor or a dedicated CSV editor.
- **Do not add or delete rows** in the translation files.
- **Do not modify or delete bracketed placeholders** such as `{...}` or `[...]` within the text. Removing them will prevent critical game information from being displayed properly.
- As a rule, translations will be merged based on trust.
- **Partial contributions are completely fine!** Even translating a single line is highly appreciated.
- Machine translations (AI/DeepL/etc.) are acceptable, but please ensure a **human final check** is done before submission to maintain natural phrasing.
